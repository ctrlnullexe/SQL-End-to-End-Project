This folder focuses on ranking entities based on measures in the data warehouse.

### Purpose
- Rank top/bottom customers, products, or categories by sales, quantity, or revenue.  
- Identify high-impact performers and areas needing attention.  
- Provides insights for dashboards, KPIs, and business decisions.

### Key Insights
- Certain products or customers dominate sales, highlighting Pareto effects.  
- Rankings help prioritize marketing, inventory, or analytical focus.  
- Rankings complement magnitude analysis for a full picture of performance.

### How to Use
1. Open `ranking_analysis.sql`
2. Run queries against the `gold` layer.  
3. Use inline comments to understand top/bottom performers and patterns.
